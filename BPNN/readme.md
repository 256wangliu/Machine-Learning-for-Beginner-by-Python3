#### 理论推导

+ **符号说明**

    1. 神经网络的层数m，也就是包括m-2个隐层；
    2. 输入层定义为I，节点数为样本的输入维度数ni；
    
       隐层定义为Hh，h为1到m-2，每一个隐层的节点数为Nh；
       
       输出层为O，节点数为样本的输出维度数No；
       
    3. 每一个隐层的权重为Ws，s为0到m-3，ws矩阵的大小为g\*t,g为该隐层前一层的节点数，t为该隐层的节点数；
    
       对应的偏置为Bs，s为0到m-3，ws矩阵的大小为1\*t, t为该隐层的节点数；；
